### Usage
Pour lancer le serveur, exécuter la commande suivante : ``` node server.js ``` ou ```foreman start web```  

Pour démarrer le relais Arduino <--> serveur, éxécuter les commandes suivantes dans l'ordre :  
```
cd arduino_server/  
node app.js
```
