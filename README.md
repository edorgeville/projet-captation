### Usage

Pour lancer le serveur, exécuter les commandes suivantes :
```
npm install
node server.js
```

Pour démarrer le relais Arduino <--> serveur, éxécuter les commandes suivantes dans l'ordre :  
```
cd arduino_server/
npm install
node app.js
```
